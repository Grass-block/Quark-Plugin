# WE选区渲染 <Badge type="tip">we-session-renderer</Badge>

文: GrassBlock2022

-----

> 需要插件: `WorldEdit` 任意

可视化你的WorldEdit选区。提供多种渲染模式。

### 命令:
渲染选区: `/render-we-selection`,`//render`,`/render-selection`

- `<无参数>` | `render` 立刻暂时渲染你的WE选区
- `persistent` 切换渲染模式为 [持久]
- `update` 设置渲染模式为 [更新]
- `off` 关闭渲染
